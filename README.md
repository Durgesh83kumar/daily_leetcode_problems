# LeetCode Daily Practice

This repository contains daily LeetCode practice problems solved in Python.

## Goal

- Upload one LeetCode problem solution per day.
- Practice algorithm and data structure problems consistently.
- Build a clean repository with clear problem names and explanations.

## Structure

- Each file is named with the LeetCode problem number and title.
- Solutions are written in Python.

## Example

- `11. Container With Most Water.py`

## How to Use

1. Add a new Python file for each problem.
2. Name files as `{problem number}. {problem title}.py`.
3. Include a short comment or docstring at the top with the problem description or approach.
4. Commit and push daily to GitHub.

## Tips

- Keep solutions readable and well-formatted.
- Add test cases or sample input/output when helpful.
- Optionally include a `solutions/` folder if the repository grows larger.
